# HeartGuard – Heart Disease Prediction System

## Introduction
* HeartGuard is a machine learning–based web application designed to predict the likelihood of heart disease in patients based on medical attributes. The system focuses on delivering accurate predictions along with clear insights to assist users in understanding their health condition.
* The project emphasizes manual implementation of machine learning algorithms, ensuring a deeper understanding of core concepts without relying on external ML libraries.

## Purpose
* To provide early prediction of heart disease risk
* To assist users and healthcare learners in understanding key health indicators
* To demonstrate end-to-end ML pipeline development from scratch


## Programming Language

### Python
* Core language for implementing the entire ML pipeline and backend logic
* Used for data processing, algorithm development, and model evaluation

### Machine Learning (Manual Implementation)
* Logistic Regression
* K-Nearest Neighbors (KNN)
* Decision Tree
* Grid Search (manual tuning)

### Data Handling
* CSV file processing (manual parsing)
* Data normalization and encoding
* Feature engineering
  
### Visualization & Explainability
* SHAP (model interpretability)
* LIME (local explanations)

### Frontend & Deployment
* Streamlit (UI development)
* GitHub (version control)
* Streamlit Cloud (deployment)

### Current Features
* Heart disease prediction using multiple ML models
* Manual implementation of algorithms (no sklearn training)
*  Interactive Streamlit UI (Spotify-style layout)
*  Model comparison and performance evaluation
*  feature importance visualization using SHAP
*  cal prediction explanation using LIME
*  lean navigation: Home, Prediction, Insights
*  reprocessing pipeline (normalization, encoding)

## Live Link of the project
[Open HeartGuard App](https://heartguard-app.streamlit.app/)

## Conclusion
* HeartGuard demonstrates a complete machine learning workflow, from raw data handling to deployment, with a strong emphasis on manual implementation and interpretability. It serves as both a practical healthcare tool and a learning-focused project for understanding ML fundamentals.
