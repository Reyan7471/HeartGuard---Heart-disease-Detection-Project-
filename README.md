❤️ HeartGuard – Heart Disease Detection System

A fully manual machine learning system for predicting heart disease risk, built from scratch without external ML libraries and deployed as an interactive web application.

⚠️ This application is intended for educational purposes and should not be used as a substitute for professional medical advice.

📌 Project Overview

HeartGuard is an end-to-end machine learning system that predicts the likelihood of heart disease based on clinical input features.

This project is intentionally designed to avoid high-level ML libraries and instead focuses on:

Manual implementation of algorithms
Full control over preprocessing and training
Transparent and interpretable predictions

Objectives
Build a heart disease prediction system from scratch
Avoid reliance on scikit-learn or similar ML frameworks
Implement optimization and tuning techniques manually
Provide interpretability using SHAP and LIME
Deploy a production-ready web application
- Key Features
- Prediction Engine
Accepts medical parameters from users
Outputs:
Risk classification (High / Low)
Probability score

- Interpretability Layer
SHAP → global feature importance
LIME → local prediction explanations

- User Interface
Spotify-inspired layout
